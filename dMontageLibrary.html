Over 750 professional Effects. This is a library of 750 effects and presets which will save you a lot of time and help you professionally edit your video in a matter of minutes. Montage Library contains all types of ready to use effects from 17 different categories: Camera Movements, Chromatic Effects, Color Effects, Distortion, Light Leaks, Screen Beats, Speed Beats, Time Effects, Transitions, Color Correction, Screen Effects. Every category contains different types of effects and creates unique stylish look. Apply a preset in one click and use Control Panel to adjust it like you want. We will frequently update Montage Library and add more categories and effects.

How does this work? (Or watch tutorial in package)

Montage Library filder - MUST BE in the same folder from we instal .panel file
Installation:
 
1) change extension .zxp file to .zip
2) unpack .zip archive to AnimationStudio folder (it must be created by us)
C://program files (86)/common files/ Adobe/ cep/extensions/AnimationStudio
3) copy entire MontageLibrary folder to ANimationSTudio Folder (C://program files (86)/common files/ Adobe/ cep/extensions/AnimationStudio
4) copy .panel file to ANimationSTudio Folder C://program files (86)/common files/ Adobe/ cep/extensions/AnimationStudio
5) Lunch AE, Lunch Animation Studio, click "d" letter then "Instal" and choose .panel file
